With this command, the HL-driver is told to signal incoming calls for the given EAZs/MSNs to the LL.

Parameter:
driverdriver-Id.
commandISDN_CMD_SETEAZ
argchannel-number locally to the driver. (starting with 0)
para.num ASCII-String, containing the desired EAZ's/MSN's (comma-separated). If an empty String is given, the HL-driver should respond to ALL incoming calls, regardless of the destination-address.

***CHANGE0.6: New since this version the "empty-string"-feature.
